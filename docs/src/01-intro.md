# Введение в Rust4ui

**Rust4ui** (Rust for UI) — фреймворк для быстрого прототипирования и сборки интерфейсов на базе `egui` 0.32. Управление UI происходит через JSON-файлы, а логика пишется на Rust.

## Три слоя архитектуры

```
┌─────────────────────────────────────┐
│  ui.json  — структура и виджеты      │
├─────────────────────────────────────┤
│  theme.json — цвета, размеры, стили  │
├─────────────────────────────────────┤
│  Rust код — экшены, состояния,       │
│  бизнес-логика                       │
└─────────────────────────────────────┘
```

### 1. UI JSON (`ui.json`)
Описывает дерево виджетов: тип, атрибуты, вложенные элементы. Каждый узел содержит поле `type` и набор атрибутов. Поддерживаются комментарии `//` и `/* */`.

### 2. Theme JSON (`theme.json`)
Определяет визуальное оформление: цвета, размеры, скругления, отступы для каждого типа виджета. Тема загружается поверх дефолтной тёмной темы и переопределяет её ключи.

### 3. Rust actions (логика)
Экшены регистрируются в `ActionRegistry` и вызываются из JSON по имени. Состояние хранится в `StateRegistry` и автоматически синхронизируется с UI через binding.

## Для кого

- Разработчики, уставшие писать UI на Rust вручную
- Прототипирование интерфейсов без перекомпиляции
- Инструменты, где нужен динамический UI (DAW, редакторы, мониторинг)
- Команды, где дизайнеры правят JSON, а разработчики — логику

## Ключевые возможности

- **30+ виджетов**: Column, Row, Label, Button, TextField, Checkbox, RadioGroup, Slider, ComboBox, Tabs, Panel, ScrollArea, Window, Spinner, Shortcut, ColorPicker, FileDrop, Indicator, IconButton, IconBar, Caption, StatusBar, Hyperlink, Grid, MenuBar, Menu, MenuItem, SubMenu, Notifications, Separator, Spacer
- **1512 иконок Phosphor** — вкомпилированы в бинарь, доступны по реальным именам
- **Hot-reload** — файловый watcher перезагружает ui.json и theme.json на лету
- **Локализация** — `{{key}}` в тексте, CLDR plural rules для 10 языков
- **Темы** — dark/light/dj_green, приоритет: атрибут виджета > тема > egui default
- **Границы** — solid/dash/dot, с позицией inside/center/outside
- **Тени** — offset, blur, color
- **Hover/Click состояния** — per-widget и через тему
